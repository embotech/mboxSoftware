## incaslib

Contains the INCAS communication library.

The expected directory tree is shown below:

incaslib
├── CMakeLists.txt
├── README.md
├── arm
│   └── libincas.a
├── include
│   └── incaslib.h
└── intel
    └── libincas.a

The last tested version is v2.5. For more information contact [incas@incas.ro](mailto:incas@incas.ro).
